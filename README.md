# AT89S52 Radar System

A radar-style distance monitoring system built using the AT89S52 microcontroller and Assembly Language.

## Features
- HC-SR04 distance measurement
- Real-time LCD display
- RGB LED status indication
- Buzzer alerts for close objects
- Anti-flicker LCD updates

## Hardware
- AT89S52
- HC-SR04
- 16x2 LCD
- RGB LED
- Buzzer

## System Logic
1. Trigger sensor
2. Measure echo pulse
3. Calculate distance
4. Update LCD
5. Activate alerts
